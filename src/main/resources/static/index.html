El problema

El problema

Para una matriz de 3 dimensiones, inicializada con 0 (cero) en todas sus celdas, se debe dar soporte para dos operaciones: UPDATE y QUERY.

  • UPDATE x y z W
    • 'x', 'y', 'z' son las coordenadas dentro del cubo
    • 'W', el valor que se le asigna en el update.
  • QUERY x1 y1 z1 x2 y2 z2
    • 'x1', 'y1', 'z1' son las coordenadas de la primer celda del rango a sumar
    • 'x2', 'y2', 'z2' son las coordenadas de la última celda del rango a sumar
